# MERN Portfolio Starter – Ready-to-Run Boilerplate

## Quick Start
```bash
# Backend
cd backend
cp .env.example .env
npm install
npm run dev  # http://localhost:5000

# Frontend (new terminal)
cd ../frontend
npm install
npm start    # http://localhost:3000
```
Set `frontend/.env` if you deploy backend elsewhere:
```
REACT_APP_API_BASE_URL=http://localhost:5000
```
